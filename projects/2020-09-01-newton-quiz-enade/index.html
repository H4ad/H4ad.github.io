Newton Quiz Enade is a mobile app to help students study for the annual quiz called Enade. In this app, students can answer questions they receive daily and also challenge other students with some questions.

In this project we use Angular, TypeScript and Capacitor to develop the mobile app. For the API, we use NestJS and TypeScript and deploy on AWS Lambda behind an AWS API Gateway HTTP, using AWS RDS as a database service.
