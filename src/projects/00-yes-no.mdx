YesNo é um aplicativo para você enviar dúvidas em grupos de conversas, ou diretamente para uma pessoa,
para coletar opiniões. Com ele, você consegue ver estatísticas sobre o que as pessoas escolheram, comentários e outros.

Nesse projeto eu fui responsável por desenvolver a programação do aplicativo, assim como, a partir de Designs, criar todas as telas.
