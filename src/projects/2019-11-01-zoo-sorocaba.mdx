The Municipal Zoological Park Quinzinho de Barros brings a great adventure to the palms of your hands.
The PZMQB application brings a lot of information about the animals present at the Sorocaba Zoo in a very fun and interactive way.

In this project I was responsible for developing the mobile app in `Angular 2` and `Ionic 4`, using `Cordova` to build the application for the store.
All the code was written in `TypeScript`, using `SCSS` as the style language organizing in the methodology [BEM](https://getbem.com/introduction/).
