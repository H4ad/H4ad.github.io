Nesse projeto foi desenvolvido uma Skill para a Alexa com a temática do Show do Milhão,
aonde o usuário pode ter uma experiência ao programa que marcou tanto sua época.

As tecnologias utilizadas nesse projeto foram TypeScript, na qual foi usada para desenvolver
a Skill que lida com todo a experiência. Além disso, foi desenvolvido uma API com ASP .NET Core
para fornecer as questões para a Skill, e tudo isso foi hospedado em uma instância no EC2.
