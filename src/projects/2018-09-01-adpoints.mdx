Adpoints is a mobile application like Google Reward which aims to give user points for watching ADS and answering some questions about ADS.|]

In this project I was responsible for developing the application in `Angular 2` and `Ionic 3`, using `Cordova` to build the application for the store.
All the code was written in `TypeScript`, using `SCSS` as the style language organizing in the methodology [BEM](https://getbem.com/introduction/).
