Esse é um plugin no Cordova, que permite você compartilhar Stickers ( figurinhas ) para o seu WhatsApp a partir
de um aplicativo criado com o Ionic. Ele foi criado a partir da necessidade de se compartilhar as figurinhas diretamente do Ionic.

Na época, eu não encontrei nenhuma documentação ou código a respeito, apenas código nativo ( Java ou Ios ),
então eu decidi criar por mim mesmo um plugin para fazer justamente essa ponte entre o código nativo e o TypeScript.
