Ingenico is a merchant services technology company based in France. In this project, an application was developed to work inside an intelligent terminal.
The customer, waiter and manager can use the application to order food, manage receipts or control inventory, all done within the smart terminal.
In addition, in this project some `Capacitor` plugins were developed to be able to use the payment and printing SDK of the smart terminal.

In this project I was responsible for developing the mobile app in `Angular 2` and `Ionic 4`, using `Capacitor` to build the application.
All the code was written in `TypeScript`, using `SCSS` as the style language organizing in the methodology [BEM](https://getbem.com/introduction/).
